### Project Overview
[cite_start]Developed an interactive scouting dashboard designed to bridge the gap between complex analytics and actionable player evaluation. The tool allows users to visualize and compare player metrics dynamically.

![NFL Scouting Dashboard](/images/your-dashboard-screenshot.png)

### Technical Execution
* **Data Visualization:** Built using R and Shiny to create a responsive, user-friendly interface for exploring player data.
* **Evaluation Metrics:** Aggregated and visualized key performance indicators to assist in identifying high-value targets and evaluation trends.
* **Skills:** R, Shiny, Data Visualization, Sports Analytics
* **Live App:** [Interactive Shiny Dashboard](https://jakeblumengarten.shinyapps.io/NFL_Scouting_Dashboard/)
