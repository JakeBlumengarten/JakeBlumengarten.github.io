### Project Overview
Official submission to the NFL's Big Data Bowl 2025 - Prediction Competition. The core objective was to predict player movements and end positions on passing plays.

### Technical Execution
* Leveraged tracking data from the 2023 NFL season.
* Built and trained machine learning models (**XGBoost**) using pre-throw player tracking data to predict post-throw movement patterns (dx, dy).
* Generated animated play visualizations directly from the tracking dataset to illustrate predicted outcomes.
* **Skills:** Data Analysis, Python (Machine Learning), Spatial Tracking
